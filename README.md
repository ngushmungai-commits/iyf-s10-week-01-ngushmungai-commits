## ğŸš€ Getting Started

### Prerequisites
- A modern web browser (Chrome, Firefox, Safari, Edge)
- A code editor (VS Code, Sublime Text, etc.) - optional
- Your profile image named `GEEK.jpg`

### Installation

1. **Download all files** and place them in the same folder
2. **Add your profile image** - Replace `GEEK.jpg` with your own image or rename your image to `GEEK.jpg`
3. **Open `index.html`** in your web browser

### Customization

#### Update Personal Information
- Open each HTML file in a text editor
- Replace "Wayne Mungai" with your name
- Update email, phone, and location in `contact.html`
- Modify skills percentages in `about.html`

#### Add Your Projects
- Edit `projects.html`
- Replace placeholder projects with your actual projects
- Update project links to point to your demos and GitHub repositories

#### Change Colors
The site uses a color scheme based on:
- **Primary**: Indigo (`indigo-900`, `indigo-800`)
- **Accent**: Pink (`pink-300`, `pink-400`, `pink-500`)

To change colors, search and replace the Tailwind color classes throughout the HTML files.

#### Update Social Links
In `contact.html`, update the social media links:
```html
<a href="YOUR_GITHUB_URL">GitHub Icon</a>
<a href="YOUR_LINKEDIN_URL">LinkedIn Icon</a>
<a href="YOUR_TWITTER_URL">Twitter Icon</a>
```
